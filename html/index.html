Ahagon Cheatsheet

Commands

  • r+ (SHA): Accept a PR.
  • r=NAME (SHA): Accept a PR on the behalf of NAME.
  • r-: Unaccept a PR.
  • p=NUMBER: Set the priority of the accpeted PR.
  • ...

Examples

  • @ahagon r+ p=1
  • @ahagon r+ 123456
  • @ahagon r=mikopits rollup
  • @ahagon retry
